## Overview

Balance fetch functions allow you to retrieve token balances for SOL, ETH, and NEAR. SOL and ETH balances are fetched from the NEAR contract using view methods, while NEAR balance is fetched directly from the NEAR blockchain via RPC.

<Note>
All balances return strings in the smallest unit:
- SOL: 9 decimals
- ETH: 18 decimals  
- NEAR: 24 decimals (yoctoNEAR)
</Note>

## SOL Balance

Fetch SOL balance from the NEAR contract using the `mt_balance_of` method.

<ParamField body="account_id" type="string" required>
  NEAR account ID
</ParamField>

<ParamField body="token_id" type="string" required>
  Token identifier: `"nep141:sol.omft.near"`
</ParamField>

### Function Details

- **Function**: `viewMethod` (NEAR contract view call)
- **Contract**: `intents.near`
- **Method**: `mt_balance_of`
- **Returns**: Balance as string (in smallest unit, 9 decimals)

### Code Example

```javascript
const solBalance = await viewMethod({
  contractId: "intents.near",
  method: "mt_balance_of",
  args: {
    account_id: accountId,
    token_id: "nep141:sol.omft.near",
  },
});
```

## NEAR Balance

Fetch NEAR balance directly from the NEAR blockchain using RPC query.

<ParamField body="account_id" type="string" required>
  NEAR account ID
</ParamField>

<ParamField body="finality" type="string">
  Query finality: `"optimistic"` or `"final"` (default: `"optimistic"`)
</ParamField>

### Function Details

- **Function**: `provider.query` (NEAR RPC query)
- **Provider**: `JsonRpcProvider` from `near-api-js`
- **Request Type**: `view_account`
- **Returns**: Account state object with `amount` property (in yoctoNEAR, 24 decimals)

<Warning>
If native NEAR balance query fails, the function falls back to wrapped NEAR using `mt_balance_of` with token_id: `"nep141:wrap.near"`
</Warning>

### Code Example

```javascript
const provider = new providers.JsonRpcProvider({
  url: "https://rpc.mainnet.near.org"
});

const accountState = await provider.query({
  request_type: "view_account",
  account_id: accountId,
  finality: "optimistic",
});

const nearBalance = accountState.amount; // Returns yoctoNEAR as string
```

## Summary

| Token | Function | Contract/Method | Decimals | Return Type |
|-------|----------|----------------|----------|-------------|
| SOL | `viewMethod` | `intents.near` / `mt_balance_of` | 9 | String |
| ETH | `viewMethod` | `intents.near` / `mt_balance_of` | 18 | String |
| NEAR | `provider.query` | RPC / `view_account` | 24 | String (yoctoNEAR) |

<Note>
- SOL and ETH balances are fetched from the intents contract (multi-token balances)
- NEAR balance is fetched directly from the NEAR blockchain via RPC
- All balances return strings in the smallest unit
</Note>
